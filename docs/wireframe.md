# **UI/UX 와이어프레임 가이드**

> 피그마에서 직접 디자인할 수 있도록 상세 가이드
> 

---

## **📐 기본 설정**

### **프레임 크기**

```
모바일: 390 x 844 (iPhone 14 기준)

```

### **컬러 팔레트 (제안)**

| **용도** | **색상** | **HEX** |
| --- | --- | --- |
| Primary | 그린 (비건/자연) | `#2D7A4F` |
| Primary Light | 연한 그린 | `#E8F5E9` |
| Secondary | 베이지 (빵/따뜻함) | `#F5E6D3` |
| Accent | 오렌지 (CTA 버튼) | `#FF6B35` |
| Text Primary | 다크 그레이 | `#1A1A1A` |
| Text Secondary | 미디엄 그레이 | `#6B6B6B` |
| Background | 화이트 | `#FFFFFF` |
| Surface | 라이트 그레이 | `#F8F8F8` |
| Error/품절 | 레드 | `#E53935` |
| Success | 그린 | `#43A047` |
| Warning/소진임박 | 옐로우 | `#FFA726` |

### **타이포그래피**

| **용도** | **크기** | **굵기** | **예시** |
| --- | --- | --- | --- |
| 대제목 (H1) | 24px | Bold | 앱 타이틀 |
| 중제목 (H2) | 20px | SemiBold | 섹션 제목 |
| 소제목 (H3) | 16px | SemiBold | 카드 제목 |
| 본문 | 14px | Regular | 설명 텍스트 |
| 캡션 | 12px | Regular | 보조 정보 |
| 가격 | 18px | Bold | ₩3,000 |

> 추천 폰트: Pretendard 또는 Noto Sans KR
> 

### **간격 시스템**

```
4px  - 아이콘 내부
8px  - 요소 간 최소 간격
12px - 카드 내부 패딩
16px - 섹션 간 간격
20px - 화면 좌우 마진
24px - 큰 섹션 구분

```

## **🔥 킬러 기능: 빵드롭 (Bread Drop)**

> 핵심 컨셉: "갓 구운 빵을 잡아라!" - 빵이 나오는 순간을 이벤트로 만들어 단순 주문이 아닌 경험으로 전환
> 

### **컬러 추가 (빵드롭 전용)**

| **용도** | **색상** | **HEX** |
| --- | --- | --- |
| 드롭 타이머 배경 | 딥 오렌지 그라데이션 | `#FF6B35` → `#FF8C42` |
| LIVE 배지 | 핫 레드 | `#FF3B30` |
| 카운트다운 숫자 | 화이트 | `#FFFFFF` |

## **🎨 피그마 작업 팁**

### **1. Auto Layout 활용**

- 모든 카드/리스트에 Auto Layout 적용
- 간격: 8, 12, 16px 일관되게

### **2. 컴포넌트화**

```
Components/
├── Buttons/
│   ├── Primary (CTA)
│   ├── Secondary
│   └── Text
├── Cards/
│   ├── ProductCard (가로/세로)
│   ├── OrderCard
│   └── SummaryCard
├── Inputs/
│   ├── TextField
│   ├── Stepper (수량)
│   └── Toggle
├── Navigation/
│   ├── TopBar
│   └── BottomTab
└── Tags/
    ├── VeganTag
    └── StockStatus

```

### **3. Variants 설정**

**ProductCard Variants:**

- State: Default / SoldOut / LowStock
- Size: Small (120x160) / Large (가로형)

**Button Variants:**

- State: Default / Pressed / Disabled
- Size: Small / Medium / Large

### **4. 프로토타입 연결**

주요 플로우:

1. 홈 → 메뉴 상세 → 장바구니 → 결제 → 완료
2. 대시보드 → 재고 관리 → 수량 변경
3. 알림 버튼 → 바텀시트 → 발송
